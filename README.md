# Home Assistant blueprint to control your IKEA four-button remote controls for lights 

This repository contains a simple Home Assistant blueprint to create automations for the IKEA Remote Control N2.  
It is a fork of 
[Malte's existing blueprint](https://community.home-assistant.io/t/zha-ikea-four-button-remote-styrbar-for-lights-e2001-e2002/384482/1)
in the Home Assistant Community Forums.

## Key differences over the existing blueprint:

- All four buttons can freely be customized
